
AWS Core Services and Their Examples
------------------------------------

1. Compute Services
-------------------
1.1 EC2 (Elastic Compute Cloud)
    - Virtual machines to run applications.
    - Example: Host a website on an EC2 instance.

1.2 Lambda
    - Serverless function execution triggered by events.
    - Example: Resize uploaded images in S3.

1.3 Elastic Beanstalk
    - Platform to deploy and manage web applications.
    - Example: Deploy a Node.js web app with no infrastructure setup.

1.4 AWS Batch
    - Run batch computing jobs at any scale.
    - Example: Process large volumes of video files.

2. Storage Services
-------------------
2.1 S3 (Simple Storage Service)
    - Object storage for files, images, backups.
    - Example: Store user profile images.

2.2 EBS (Elastic Block Store)
    - Block storage for EC2 virtual hard drives.
    - Example: Store application data on EC2.

2.3 EFS (Elastic File System)
    - Shared file storage for Linux EC2 instances.
    - Example: Share configuration files across servers.

3. Networking Services
----------------------
3.1 VPC (Virtual Private Cloud)
    - Isolated network for your AWS resources.
    - Example: Setup private and public subnets for an app.

3.2 Route 53
    - DNS and domain name service.
    - Example: Route domain traffic to EC2.

3.3 ELB (Elastic Load Balancer)
    - Distributes traffic across servers.
    - Example: Load balance traffic to your app.

3.4 CloudFront
    - Content Delivery Network (CDN).
    - Example: Deliver images and videos globally.

4. Database Services
--------------------
4.1 RDS (Relational Database Service)
    - Managed SQL databases.
    - Example: Host a MySQL database.

4.2 DynamoDB
    - NoSQL managed database.
    - Example: Store user sessions or shopping carts.

4.3 Aurora
    - High-performance SQL database.
    - Example: Backend database for SaaS.

4.4 ElastiCache
    - In-memory cache service.
    - Example: Cache frequent queries.

5. Security and Identity
------------------------
5.1 IAM (Identity and Access Management)
    - Manage user access.
    - Example: Allow read-only access to S3.

5.2 KMS (Key Management Service)
    - Encrypt data with managed keys.
    - Example: Encrypt S3 files.

5.3 AWS Shield
    - DDoS protection service.
    - Example: Protect public websites.

6. Analytics and Monitoring
---------------------------
6.1 CloudWatch
    - Monitor resources and set alerts.
    - Example: Alert on high CPU usage.

6.2 Glue
    - Managed ETL service.
    - Example: Transform data from S3 to Redshift.

6.3 Redshift
    - Data warehousing service.
    - Example: Business analytics.

7. Machine Learning
-------------------
7.1 SageMaker
    - Build and train ML models.
    - Example: Predict house prices.

7.2 Rekognition
    - Image and video analysis.
    - Example: Detect faces in uploaded images.

8. Developer Tools
------------------
8.1 CodeCommit
    - Git repository hosting.
    - Example: Manage project code.

8.2 CodeDeploy
    - Automated deployment.
    - Example: Deploy app to EC2.

8.3 CodePipeline
    - CI/CD pipeline.
    - Example: Automate app builds and deployments.

9. Application Integration
--------------------------
9.1 SQS (Simple Queue Service)
    - Message queue for decoupling.
    - Example: Queue orders for processing.

9.2 SNS (Simple Notification Service)
    - Messaging and notifications.
    - Example: Send SMS on user signup.

9.3 Step Functions
    - Workflow orchestration.
    - Example: Coordinate Lambda functions.

10. Management and Governance
-----------------------------
10.1 CloudFormation
    - Infrastructure as Code.
    - Example: Define full architecture in a template.

10.2 CloudTrail
    - API activity tracking.
    - Example: Audit who accessed S3.

10.3 AWS Config
    - Resource configuration tracking.
    - Example: Monitor for non-compliant setups.
